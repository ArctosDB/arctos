Demo with fullwindow-plugin and useractivity-event

This demo adds the fullwindow plugin, the useractivity-event plugin and a fallback message to our standard example using jmeEmbedControls.

Difference between this demo and the normal fullwindow/useractivity demo: The width of the timeline-slider is calculated using javascript, which makes it much simpler to position each control.

Please have a look at the mediaControls - configuration.
